How It Works

The Inverse Problem

Given a trajectory, what force fields would cause a ball to follow that path? This is an inverse problem â€” we're working backwards from effect to cause.

1

Draw a Trajectory

Sketch any path you want â€” curves, spirals, zigzags. This becomes the target trajectory the model needs to reproduce.

2

Encode with Transformer

A transformer encoder processes the trajectory points, using self-attention to understand the global shape and local curvature.

3

Predict Force Fields

A slot-based decoder outputs up to 6 force fields â€” each with a type (wind, vortex, attractor, repeller) and parameters.

4

Simulate & Compare

The predicted fields are used in a differentiable physics simulation. The loss is the difference between predicted and target trajectories.

Differentiable Physics

The key insight: physics simulation is differentiable, so we can backpropagate through it.
